Aqui, mostraremos o passo a passo para inserção e formatação de imagens usando HTML e CSS.


Inserção das imagens


1° passo - Criar um arquivo html


2º passo - Criar a pasta de imagem com o nome "img" e a pasta "css"


3º passo - Baixar a imagem(ns) desejada(s) na pasta "img"


4º passo - Voltar para o arquivo HTML e na parte do "body" usar a tag "img", seguido por src="./img/[nome da imagem].[extensão]"


Formatação das imagens


1° passo - Criar um arquivo CSS.


2° passo - Criar uma classe no arquivo html com o comando class="[nome da classe]" dentro da tag desejada.


3° passo - Ir no arquivo CSS e criar a classe através do conando ".[nome da classe] {}"


4° passo - Escrever as normas de formatação dentro das chaves da classe.
